# Lillie's Treats

An e-commerce website for Lillie's Treats — a local business selling handcrafted sweets, chin chin, and cupcakes.

Built with **Next.js 16**, **TypeScript**, **Tailwind CSS v4**, **Framer Motion**, and **Zustand**.

## Getting Started

```bash
npm install
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) to view the site.

## Tech Stack

- **Next.js 16** (App Router)
- **TypeScript**
- **Tailwind CSS v4** — design tokens defined in `app/globals.css`
- **Framer Motion** — animations
- **Zustand** — cart & wishlist state
- **Google Fonts** — Playfair Display (headings) + Poppins (body)
